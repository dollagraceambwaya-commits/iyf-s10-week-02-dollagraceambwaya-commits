# iyf-s10-week-02-DollaGrace
## CSS Mastery (Week 2)

## Overview
This project is part of **Season 10 – Week 2 CSS Mastery**.  
It demonstrates how to use **CSS Flexbox and Grid** to build responsive layouts and style elements effectively.

---

## Features
- Practiced **Flexbox** for alignment and spacing  
- Practiced **CSS Grid** for layout design  
- Applied **selectors, pseudo-classes, and pseudo-elements**  
- Styled buttons, forms, and containers with modern CSS

2. Explore the styled components and layouts.  
3. Resize the browser window to see responsive behavior.
